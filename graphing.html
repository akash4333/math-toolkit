Function Syntax

Enter mathematical functions using standard notation:

  • Basic operations: +, -, *, /
  • Exponents: x^2 or x**2
  • Trig functions: sin(x), cos(x), tan(x)
  • Square root: sqrt(x)
  • Logarithms: log(x) (base 10), ln(x) (natural log)
  • Constants: PI, E

Example functions: 2*x^3 - 5*x + 3, sin(x)/x, sqrt(x^2 + 1)
